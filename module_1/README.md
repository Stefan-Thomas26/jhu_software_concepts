# Module 1 README

## Instructions
To run this application, you must cd into `jhu_software_concepts/module_1/personal_website/`

Next, run the following command:
`python3 run.py`
The application should now be running locally.

You are able to copy the following into your internet browser and view the website: `http://192.168.0.44:8080`

## Comments regarding HTML and CSS templates for this assignment
